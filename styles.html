BlockQuotes

This is a block quote. It will also have an author cited in a nested footer.

Abraham Lincoln

Block Quotes should only be used for quotes, either within a story or by an outside source, not just for indenting text. To add a citation to a block quote, wrap a paragraph in a cite tag like this: <cite><p>Author cited</p></cite>. It's also an optional best practice to nest this citation inside a footer.
